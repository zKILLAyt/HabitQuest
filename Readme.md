# 🚀 HabitQuest

HabitQuest é uma aplicação web desenvolvida com HTML, CSS e JavaScript que transforma a construção de hábitos em uma experiência gamificada.

O objetivo do projeto é incentivar a consistência e a produtividade através de um sistema de níveis, experiência (XP) e sequência de dias consecutivos (Streak).

## 🎮 Sistema de Progressão

Cada hábito concluído concede:

```text
+10 XP
```

Os níveis são calculados de forma progressiva:

```text
Nível 1 → 100 XP
Nível 2 → 200 XP
Nível 3 → 300 XP
Nível 4 → 400 XP
...
```

Quando o jogador acumula XP suficiente, ele sobe de nível automaticamente.

---

## 🔥 Sistema de Streak

O HabitQuest acompanha a quantidade de dias consecutivos em que o usuário conclui pelo menos um hábito.

Exemplo:

```text
Dia 1 → 🔥 1
Dia 2 → 🔥 2
Dia 3 → 🔥 3
```

Caso um dia seja perdido, a sequência é reiniciada.
